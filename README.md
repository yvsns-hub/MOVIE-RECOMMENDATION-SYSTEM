# 🎬 CineMatch — Movie Recommender System

Content-based movie recommender using TF-IDF + BERT embeddings, mood/genre/rating filters, and TMDB Live API for new releases.

---

## Setup (run once)

```bash
# 1. Create virtual environment
python -m venv venv
venv\Scripts\activate          # Windows
source venv/bin/activate       # Mac/Linux

# 2. Install dependencies
pip install -r requirements.txt

# 3. Add your TMDB API key (free at themoviedb.org/settings/api)
copy .env.example .env
# Edit .env and paste your key

# 4. Download dataset from Kaggle
# https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata
# Place inside /data/:
#   tmdb_5000_movies.csv
#   tmdb_5000_credits.csv

# 5. Build the pipeline
python utils/data_loader.py    # ~5 seconds
python utils/tfidf_engine.py   # ~30 seconds
python utils/bert_engine.py    # ~5 minutes (downloads model first time)

# 6. Launch
streamlit run app.py
# OR double-click run.bat
```

```
│   ├── sim_tfidf.pkl           # Generated by tfidf_engine.py
│   ├── tfidf_vectorizer.pkl    # Generated by tfidf_engine.py
│   └── title_index.pkl         # Generated by tfidf_engine.py
└── utils/
    ├── data_loader.py          # Step 1: clean + merge dataset
    ├── tfidf_engine.py         # Step 2: TF-IDF similarity matrix
    ├── bert_engine.py          # Step 3: BERT semantic embeddings
    ├── filters.py              # Mood/genre/rating/year filters
    └── tmdb_api.py             # TMDB Live API (posters, new movies, reviews)
```

## Features

| Feature | Detail |
|---|---|
| TF-IDF engine | Keyword-based similarity, instant |
| BERT engine | Semantic similarity — understands plot meaning |
| Mood filter | Happy / Dark / Epic / Emotional / Informative |
| Genre filter | 18 genres, multi-select |
| Rating filter | Min rating slider |
| Year filter | Release year range |
| TMDB Live | New movies (post-2017) fetched automatically |
| Posters | Fetched from TMDB CDN, cached 24hrs |
| Reviews | Audience reviews for any movie |
| Trending | This week's trending movies |
| Now Playing | Current cinema releases |

## TMDB API Key

1. Create free account at https://www.themoviedb.org
2. Settings → API → Create → Developer
3. Copy **API Key (v3 auth)**
4. Paste into `.env` as `TMDB_API_KEY=your_key`
